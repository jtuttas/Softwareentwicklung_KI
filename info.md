# Begleit-Repository zur Fortbildung "KI-gestützte Softwareentwicklung"

Erstellt werden soll eine ToDo App in unterschiedlichen Komplexitätsstufen.

## Inhalte

- **PRD.md**: Enthält die Anforderungen und Wünsche an die ToDo App.
- **Lastenheft_komplex.bak**: Erweiterte Anforderungen und Wünsche an die ToDo App mit zusätzlichen Funktionen und Komplexität (Unterschiedliche Benutzer / Rechte und Datenbank).
- **SRS.md**: Beschreibung der Umsetzung und Funktionen der ToDo App mit Bootstrap und Local Storage als persistentem Speicher.
- **copilot-instruktions.bak**: Pflichtenheft für die Komplexe Anwendung als Client- / Server Architektur und SQLite Datenbank.
- **database.sqlite**: Die SQLite-Datenbankdatei für die ToDo App.
- **usability.md**: Dokumentation der Usability-Tests für die ToDo App.

## Starten der Präsentation

```

marp . --server --allow-local-files --html

```
Dann im Browser öffnen: http://localhost:8080/docs/Präsentation_Cisco.html
